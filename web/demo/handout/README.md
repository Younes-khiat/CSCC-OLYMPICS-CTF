# Participant Handout

This folder is for teams that want a small extra nudge.

## What you should know

- The app is a Spring Boot bakery site.
- Users can register, log in, claim a balance bonus, and buy products.
- The challenge objective is to obtain the VIP product reward.

## Suggested starting point

- Focus on the buy flow and the balance check.
- Compare the product price and the quantity handling.
- Think about what happens when arithmetic exceeds the normal range.
